# GAU-maps-analytics

## Описание
Аналитика для интерактивной карты уличного искусства с функцией маршрутов

## Ключевые результаты
- Анализ 6 сервисов-аналогов и пользовательских отзывов
- Определение целевой аудитории (7 сегментов) и пользовательских сценариев
- Разработка функциональных требований и MVP
- Решения для модерации и борьбы со спамом

## Моя роль
Аналитик - сбор и анализ требований, структурирование данных для визуализации объектов.
